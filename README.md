# UePay2-demo 前端快速开发模板

## 移动端案例安装
1. git clone 代码到本地
2. npm 安装
3. npm 运行

分别执行以下命令
1. `git clone https://github.com/chkui/uepay-website-demo.git`
1. `$ cd  uepay-website-demo/client`
1. `$ npm i # 必须有nodejs环境`
1. `$ npm run dev`

## 移动端说明
本按时的UI框架使用的是Antd-mobile
